# ElGgolearn 🚀

**ElGgolearn** هو منصة تعليمية تعتمد على الذكاء الاصطناعي لتحويل محتوى اليوتيوب إلى مسارات تعليمية (Roadmaps) منظمة، مما يساعد المتعلمين على الوصول لغايتهم بوضوح وسهولة.

## ✨ المميزات
- توليد مسارات تعليمية ذكية باستخدام نماذج Gemini.
- البحث الآلي عن أفضل قوائم التشغيل في اليوتيوب.
- واجهة مستخدم تفاعلية تعتمد على تقنيات الـ Web.
- إمكانية تصدير الخطط الدراسية كملفات PDF.
